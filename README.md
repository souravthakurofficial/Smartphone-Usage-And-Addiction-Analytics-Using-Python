## 📊 Project Title: Smartphone Usage & Addiction Analytics Using Python

### 🔹 Objective

Analyze smartphone usage behavior and identify patterns contributing to digital addiction using data-driven insights.

### 🔹 Tools Used

Python,
Pandas,
NumPy,
Matplo,tlib
Seaborn,
Plotly,
Jupyter Notebook,
Data Cleaning,
Feature Engineering,
Exploratory Data Analysis (EDA).

### 🔹 Key Insights
Age group 21–25 shows the highest smartphone screen time

Social media usage strongly correlates with addiction levels

Highly addicted users have significantly higher weekend usage

Increased screen time leads to higher sleep deficit

Notifications and entertainment usage drive addiction behavior
